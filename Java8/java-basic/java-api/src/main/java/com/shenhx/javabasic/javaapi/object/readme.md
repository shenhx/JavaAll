3. final理解
~~~
■final可以修饰变量，被final修饰的变量被赋初始值之后，不能对它重新赋值。

■final可以修饰方法，被final修饰的方法不能被重写。

■final可以修饰类，被final修饰的类不能派生子类。
~~~
